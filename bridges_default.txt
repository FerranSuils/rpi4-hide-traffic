# onion-pi · pool de bridges obfs4 públicos para auto-rescue

# Estos son los bridges obfs4 que vienen "de fábrica" con Tor Browser.
# Son PÚBLICOS, así que están más vigilados que los privados — útiles como
# fallback cuando los privados se han roto.
#
# El script /usr/local/sbin/onion-pi-bridge-rescue.sh:
#   1. Pide bridges frescos al Moat builtin API:
#        https://bridges.torproject.org/moat/circumvention/builtin
#   2. Si Moat no responde, usa esta lista bundled.
#   3. Testea TCP a cada candidato, se queda con los primeros 3-5 alive.
#   4. Reescribe /etc/tor/torrc y reinicia tor@default.
#
# Refrescar esta lista vía OTA cuando hagamos `git push` con bridges nuevos.
# Fuente canónica de Tor Browser:
